Toko Bintnag - Simulasi Text-Base 

📋 Deskripsi

Toko Bintnag adalah aplikasi simulasi berbasis teks (command-line interface) yang dikembangkan menggunakan Python. Aplikasi ini mensimulasikan pengalaman berbelanja di sebuah toko virtual, di mana pengguna berinteraksi dengan sistem melalui menu navigasi, mengelola keranjang belanja, memeriksa stok barang, dan melakukan transaksi pembayaran.
Proyek ini dirancang untuk mendemonstrasikan implementasi logika pemrograman berorientasi objek (OOP) sederhana, manipulasi state permainan, serta antarmuka pengguna berbasis teks dengan animasi dasar.

✨ Fitur Utama

Sistem Navigasi Lokasi: Pengguna dapat berpindah antara area "Kasir", "Rak Makanan", dan "Rak Peralatan".

Manajemen Keranjang Belanja:

Mengambil dan mengembalikan keranjang fisik.

Menambahkan barang ke keranjang dengan validasi stok.

Database Produk Dinamis:

Data produk tersimpan dalam struktur dictionary.

Mendukung kategori, varian, harga, stok, dan status promosi.

Sistem Transaksi & Pembayaran:

Perhitungan otomatis total belanja.

Penerapan diskon/promo secara real-time.

Pengurangan stok database setelah pembayaran sukses.

Interaksi NPC (Kasir):

Fitur tanya jawab mengenai ketersediaan barang dan lokasi.

Informasi promosi terbaru.

Antarmuka Visual Teks:

Animasi pintu geser saat awal permainan.

Efek pengetikan (typewriter effect) untuk dialog.

Pembersihan layar otomatis (auto-clear) untuk UX yang bersih.


🛠️ Prasyarat
Python 3.x (Disarankan versi 3.6 atau lebih baru).
Terminal atau Command Prompt (CMD/PowerShell) yang mendukung karakter Unicode.
